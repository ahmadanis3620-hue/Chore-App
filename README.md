# Cross Creek Chore Connect

A lightweight neighborhood app prototype for matching local families with people willing to help with chores.

## What it does

- Lets parents, kids, or neighbors post chore requests.
- Stores chores in browser local storage.
- Supports search and max-budget filtering so nearby helpers can quickly find work.

## Run locally

Open `index.html` directly in your browser, or run:

```bash
python -m http.server 8000
```

Then visit <http://localhost:8000>.
